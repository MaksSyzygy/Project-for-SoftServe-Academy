Доброго времени суток! Меня зовут Рыжак Максим, представляю Вашему вниманию мой проект банковского приложения.

Программа симулирует работу со стороны сотрудника банка или клиента банка

Программа написана с использованием C#, LINQ, ADO.NET, T-SQL и является консольным приложением.

Работа приложения осуществляется по принципу манипулирования данными из базы данных с дальнейшим их отображением,
изменением и сохранением.

В папке DB - БД и скриншоты таблиц с файлом readme

В папке Diplom - проект и файл readme
